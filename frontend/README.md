# DonLabs

Marketing site for DonLabs — a staff engineer-led AI consultancy helping teams ship production AI.

Built with Next.js 14 (App Router), TypeScript, and Tailwind CSS.

## Develop

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build

```bash
npm run build
npm run start
```

## Project layout

- `src/app` — App Router entry (`layout.tsx`, `page.tsx`, `globals.css`)
- `src/components` — Section components composed by `page.tsx`
